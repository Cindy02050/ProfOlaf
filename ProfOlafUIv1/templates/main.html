How It Works

  1. Upload: Upload a file containing research paper titles
  2. Configure: Set parameters like year range, venue ranks, and proxy settings
  3. Search: The tool searches Google Scholar for each paper
  4. Collect: Metadata is collected including citations, authors, venue, etc.
  5. Store: Results are stored in the SQLite database
  6. View: Browse results and database contents
Note: This process may take several minutes depending on the number of papers and network conditions.

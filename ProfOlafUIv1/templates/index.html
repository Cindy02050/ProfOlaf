Upload Research Papers

Upload a file containing research paper titles to start the snowball sampling process. The tool will search Google Scholar for each paper and collect metadata.

Supported formats: .txt (one title per line) or .json (with 'papers' array containing 'title' fields)
Year to start the search from
Year to end the search
Comma-separated list of accepted venue ranks (e.g., A,B,C)
Optional proxy key for accessing Google Scholar
Name of the final CSV output file
Delay to avoid rate limiting from Google Scholar
Path to SQLite database file
How It Works
  1. Upload: Upload a file containing research paper titles
  2. Search: The tool searches Google Scholar for each paper
  3. Collect: Metadata is collected including citations, authors, venue, etc.
  4. Store: Results are stored in the SQLite database
  5. View: Browse results and database contents
Note: This process may take several minutes depending on the number of papers and network conditions.

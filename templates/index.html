About ProfOlaf

ProfOlaf is a tool designed to help researchers with literature reviews by automating the process of snowballing articles through an initial seed.

The tool is designed to be used in a systematic literature review process. The standard workflow of the tool is described in the diagram below.

Note: currently, backward snowballing is only supported for Semantic Scholar. Google Scholar only supports forward snowballing.

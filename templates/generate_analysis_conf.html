Analysis Configuration:
  • Articles Folder: Folder where downloaded PDF files will be stored
  • CSV Path: Path to the final CSV file from the snowballing workflow (Step 10: Generate CSV)
  • Seed File: Markdown file stored in the prompts folder, serves as a seed for analysis tasks
  • Output Path: Folder where analysis results will be saved
  • Topics File: File name for topics data used in topic modeling
LLM Configuration:
  • API Keys: Leave empty to use environment variables. Otherwise, provide the API key directly.
  • Model: The model name to use for each provider (e.g., gpt-4o, gemini-pro, claude-3-sonnet-20240229)
  • Temperature: Controls randomness in outputs (0.0 = deterministic, 2.0 = very random)
  • Context Length: Maximum number of tokens the model can process in one request
